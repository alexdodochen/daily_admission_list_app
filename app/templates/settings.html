沒有 LINE Bot？點這裡看怎麼建立（一次性，~10 分鐘）
  1. 到 LINE Developers Console 用 LINE 帳號登入。
  2. 點「Create new provider」→ 名稱隨便（例：成醫心內）。
  3. 建好後點進 provider →「Create a channel」→ 選 Messaging API。
  4. 填基本資料（channel name、icon、category 隨意，region 選 Taiwan），同意條款 → Create。
  5. 進入該 channel → Messaging API 分頁：
    • 下拉到 Channel access token → 按「Issue」取得 long-lived token，貼到上面 Channel access token 欄。
    • 把該 channel 的 Bot basic ID（例如 @abc1234）加成 LINE 好友（用手機掃 QR code）。
  6. 取得 Group ID（這步最麻煩）：
    • 把 bot 加進你要推播的 LINE 群組。
    • 需要架一個 webhook 接收訊息事件 — 最快的做法：
      1. 在 Messaging API 分頁啟用「Use webhook」，URL 暫時填 https://webhook.site 給的隨機 URL。
      2. 在群組隨便發一句話，webhook.site 收到的 JSON 裡 source.groupId 就是 Group ID（C 開頭，33 字元）。
    • 把該 ID 貼到上面 Group ID 欄。
  7. 注意：channel 預設「Auto-reply messages」會擾民，建議到「Response settings」關掉。

不會做也沒關係 — ⑤ LINE 推播是選填，跳過不影響 ①-④ 的功能。

順序很重要：先按 ① 儲存 把上面填的值寫進設定檔，再按 ② 測試連線。
（測試連線跑的是已儲存的設定，不是畫面上正在編輯的值；沒先存就測，等於在測舊設定。）
